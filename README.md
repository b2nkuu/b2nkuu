## 🌟 About Me

> *"From IoT telemetry dashboards to blockchain mobile wallets — I craft digital experiences that bridge code and human intuition."*

Greetings! I'm **Nutchaitat Tantanasuwan** (but you can call me **แบงคุ** or **B2NKUU**). A software developer based in the vibrant city of **Bangkok, Thailand**, I've spent over a decade sculpting mobile applications, backend systems, and developer tools.

My journey began in the .NET era, evolved through native mobile development (Java, Swift, Xamarin), and found its true north in **Flutter** and **Golang**. Today, I'm a **Software Engineer Specialist** at Bitkub Blockchain Technology, where I help build the financial infrastructure of tomorrow.

When I'm not shipping code, you'll find me exploring Japanese philosophy (check out my **[spirit](https://github.com/b2nkuu/spirit)** project!), tinkering with tmux themes (**[city-blue](https://github.com/b2nkuu/city-blue)**), or building AI-powered tools for solopreneurs (**[solo](https://github.com/b2nkuu/solo)**).
